Introducción a Pure Vessel Library

El equipo de Pure Vessel Library realizo el siguiente proyecto con el objetivo de poder facilitar un poco el desarrollo de diseños y proyectos orientados al ámbito web, tanto para aquellas personas que trabajan de manera individual así como para las que trabajan de manera colaborativa. Dicha librería cuenta con varias utilidades que esperamos se han de ayuda para cada uno de los usuarios que hagan uso de esta.


La librería busca brindar una experiencia de usuario única mediante los conjuntos de estilos de componentes que se ofrecen para poder ser empleados en los diversos proyectos que quieran ser desarrollados. El proyecto se enfoca en el aprovechamiento de Sass el cual es un preprocesador de CSS que permite escribir estilos de manera más eficiente y modular pudiendo así definir variables, utilizar estructuras de control, crear funciones las cuales ayuden a generar estilos de forma dinámica, para posteriormente ser compilado en un archivo CSS que pueda ser utilizado para el proyecto que se tenga en mente.



Entre algunas de las cosas que podemos destacar de la librería son las distintas variedades de estilos de componentes como buttons, alertas, badges, form entre otros así también como las distintas clases utilitarias que se han creado para poder brindar de una mejor ayuda a la hora de estar programando una página web.
